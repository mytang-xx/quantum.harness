# peps-based-algorithm methodology references

No PEPS-based primary sources have been rendered into the knowledge base yet.

Before claiming reproduction of an original PEPS-based paper, use `download-ref` to
ingest the selected paper into this directory, then run `/verify` against the
paper-derived protocol. Raw PDFs, metadata, and extracted figures must stay in
local-only `.raw/` / `.figures/` subfolders.
