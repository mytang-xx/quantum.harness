# Magic / Nonstabilizerness Conventions

Harness-default conventions for Pauli-string operators, stabilizer Rényi entropies (SREs), partition modes, and qudit generalizations. Skills cite this card; they never hardcode definitions.

Cited by `knowledge-base/physics/magic/PHYSICS.md` and the model cards carrying a magic branch.

## Pauli operators

For `N` qubits (`d=2`), the Pauli group `P_N` is generated by Pauli matrices `σ^{x,y,z}` on each site, with overall phases `±1, ±i`:

```
P_N = { e^{iθπ/2} σ_{j_1} ⊗ ⋯ ⊗ σ_{j_N} : θ, j_k ∈ {0,1,2,3} }
```

For `d > 2` qudits, the Pauli operators are no longer Hermitian; expectation values may be complex and the SRE is defined on `|Tr(ρP)|`. The qudit Pauli operators are built from the shift `X` and clock `Z`:

```
X = Σ_{k=0}^{d-1} |k+1⟩⟨k|        (mod d)
Z = Σ_{k=0}^{d-1} ω_d^k |k⟩⟨k|     ω_d = e^{2πi/d}
T_{a,a'} = ω_d^{-2^{-1} a a'} Z^a X^{a'}     a, a' ∈ Z_d
```

For qutrits (`d=3`), `2^{-1}` is the inverse of 2 in Z_3 (i.e., 2). The qubit Pauli matrices recover at `d=2`.

## Stabilizer Rényi entropies (SREs)

For a pure state `ρ = |ψ⟩⟨ψ|` on `N` qudits, the order-`n` SRE is

```
M_n(ρ) = (1 / (1 − n)) log [ Σ_{P ∈ P_N} |Tr(ρP)|^{2n} / d^N ]
```

Properties:

1. *Faithfulness*: `M_n(ρ) = 0` iff `ρ` is a stabilizer state.
2. *Stability under Clifford*: `M_n(C ρ C†) = M_n(ρ)` for `C` in the Clifford group.
3. *Additivity*: `M_n(ρ_A ⊗ ρ_B) = M_n(ρ_A) + M_n(ρ_B)`.

`M_n` is a strict magic monotone (under stochastic free operations) only for `α ≥ 2`. For `0 < n < 2`, `M_n` violates monotonicity under measurements followed by conditioned Clifford transformations — it is still a useful diagnostic (and `M_1` is exactly the order-1 limit) but is not a strict monotone. **Frontier flag**: report this caveat whenever a workflow uses `M_α` with `α < 2` to draw a monotonicity-style conclusion.

The SRE *density* `m_n = M_n / N` is the canonical reported quantity for many-body states (typically linear in `N`).

For mixed states, the standard normalization is

```
M̃_2(ρ) = − log [ Σ_P |Tr(ρP)|^4 / Σ_P |Tr(ρP)|^2 ]
```

apart from an offset. Free states for the mixed-state extension are partial traces of pure stabilizer states.

## Long-range magic

For two disjoint subsystems `A` and `B` of a state `ρ_AB`,

```
L(ρ_AB) = M̃_2(ρ_AB) − M̃_2(ρ_A) − M̃_2(ρ_B)
```

`L` quantifies the magic stored in correlations between `A` and `B`. It is mutual-information-like: vanishes for a product state `ρ_A ⊗ ρ_B`, free of UV divergences in the field-theory limit, and a useful critical diagnostic.

`L(ρ_AB)` is **not sign-definite** because SRE does not satisfy subadditivity. Negative extrema at criticality are physical, not a bug.

A useful decomposition for sampling is

```
L(ρ_AB) = I_2(ρ_AB) − W(ρ_AB)
W(ρ_AB) = − log [ ⟨ |Tr(ρ_A P_A)|^4 |Tr(ρ_B P_B)|^4 / |Tr(ρ_AB P_AB)|^4 ⟩_{Π_{P_AB,2}} ]
```

with `I_2` the Rényi-2 mutual information; this decomposition allows estimating `L` directly without summing the inefficient `M̃_2` pieces separately.

## Partition modes

Three standard partition primitives appear throughout magic workflows:

1. **Full partition** — the whole system; estimator targets `M_n(ρ_full)`.
2. **Disjoint bipartite** — two widely separated subsystems `A, B`; estimator targets `L(ρ_AB)`. Standard 1D ring choice: `A = {1,…,L/4}`, `B = {L/2+1,…,3L/4}`. The choice is generic; replace by lattice analogues for 2D and other geometries.
3. **Halving increment** — `c_N = 2 M_n(N/2) − M_n(N)` cancels the volume-law term in 1D. In higher dimensions, the analogous combinations cancel volume + area-law terms (the same family of subsystem-difference schemes used to extract topological entanglement entropy).

## Qudit generalizations

| Local dimension `d` | Pauli group | Notes |
|---|---|---|
| 2 (qubit) | `{I, X, Y, Z}^N` | Standard `σ^a` matrices. |
| 3 (qutrit) | `{T_{a,a'}}` per site, `9^N` strings total | Used in `knowledge-base/models/potts-clock/MODEL.md` (3-state Clock = 3-state Potts after duality on the 1D chain). |
| `d > 3` | `T_{a,a'}` qudit Pauli per site, `d^{2N}` strings | Generic; q-state clock model. |

T-states and canonical magic states:

| Local dim | Magic state |
|---|---|
| `d=2` qubit | `|T⟩ = (|0⟩ + e^{iπ/4} |1⟩)/√2` (i.e., `|ψ(π/4)⟩` in the family `|ψ(θ)⟩ = (|0⟩ + e^{iθ}|1⟩)/√2`). |
| `d=3` qutrit | `|T⟩ = |φ(2π/9)⟩` in the family `|φ(θ)⟩ = (|0⟩ + e^{iθ}|1⟩ + e^{−iθ}|2⟩)/√3`. |

For a product state of `N` copies, `M_n(|T⟩^{⊗N}) = N M_n(|T⟩)` by additivity.

## Wegner-duality preservation

For 2D models related by Wegner duality (e.g., 2D `Z_2` lattice gauge theory ↔ 2D transverse-field Ising on the dual square lattice), the SREs are *preserved* by the duality transformation. Concretely: computing `M_n` on the dual 2D Ising ground state is equivalent to computing it on the gauge-theory ground state in the charge-free sector. Skills exploit this to drive 2D gauge-theory calculations through the (computationally more convenient) 2D Ising route.

This is a *derivation result*, not a runtime arg — it lives here once and is cited from `knowledge-base/models/transverse-field-ising/MODEL.md`'s 2D row and from `knowledge-base/physics/confinement/PHYSICS.md`.

## Symmetry-restricted Pauli proposals

When the target state is invariant under a Z_n symmetry generated by `Π_i Z_i`, the symmetry-preserving Pauli strings are generated by `Z_i` and `X_i^† X_j`. This restricts the Markov chain to symmetric Pauli strings via two-site updates.

Additional restrictions:

- For real-amplitude states (`d=2` with time-reversal symmetry), Pauli strings with an odd number of `Y` operators have zero expectation value and may be projected out before sampling.
